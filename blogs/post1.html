2. Installing Packages

Installing new software on Arch Linux is simple with the pacman package manager. For example, to install Git, use the following command:


# Install Git
sudo pacman -S git

Refer to the official documentation for a full list of packages available for Arch Linux.

3. Removing Packages

To keep your system clean, it‚Äôs important to remove software you no longer need. Use the command below to remove a package:

4. Cleaning the Package Cache

Over time, the package cache can take up a lot of space. Clean it up with this command:


# Clean package cache
sudo pacman -Sc
